# Research Note: Eco-Evolutionary Rescue and Spatial Bistability

## Executive Summary

Fixed-defense spatial patterning did not produce robust predator rescue. The fixed-defense Roy-style PDE produced one weak candidate, but it did not survive validation and mechanism checks.

Prey defense evolution produced indirect evolutionary rescue in the well-mixed ODE.

The first threshold comparison suggested a lower PDE rescue threshold than the ODE, but later stability, continuation, and basin analyses showed that a single scalar PDE-evo threshold is not well-defined in the sensitive stress range. The spatial PDE exhibits hysteresis and bistability, with predator-persistent and predator-extinct outcomes both reachable at the same stress depending on initial condition or continuation path.

ODE prey defense evolution supports indirect evolutionary rescue. In the spatial PDE, the response cannot be reduced to a single rescue threshold because the system shows hysteresis and bistability: predator-persistent and predator-extinct outcomes can both be reachable at the same stress depending on initial condition or continuation path.

## Scientific Question

Does spatial structure amplify, suppress, or qualitatively alter indirect evolutionary rescue generated by prey defense evolution?

The initial threshold quantities were:

```text
m_c^{ODE,no evo}
m_c^{ODE,evo}
m_c^{PDE,no evo}
m_c^{PDE,evo}
```

The initial derived threshold differences were:

```text
Delta_evo_ODE = m_c^{ODE,evo} - m_c^{ODE,no evo}
Delta_evo_PDE = m_c^{PDE,evo} - m_c^{PDE,no evo}
Delta_spatial_evo = m_c^{PDE,evo} - m_c^{ODE,evo}
Delta_spatial_no_evo = m_c^{PDE,no evo} - m_c^{ODE,no evo}
```

Those quantities remain useful for the ODE and for early PDE screening, but the later PDE analyses show that the spatial eco-evolutionary response cannot be summarized by a single scalar `m_c^{PDE,evo}` in the mapped stress range.

## Fixed-Defense Spatial Model

Step 07 tested the fixed-defense Roy-style spatial PDE using predator mortality thresholds. The fixed-defense PDE did not show robust `ODE extinct, PDE persistent` behavior. The best weak candidate was the `D_w/D_u=150` case, but Stage D grid escalation did not validate it, so no fixed-defense spatial rescue claim was made.

Step 08 tested whether the `D_w/D_u=150` candidate had a real spatial-growth mechanism. The mechanism diagnostic was:

```text
spatial_growth = mean(w * A)
meanfield_growth = mean(w) * A(mean fields)
spatial_covariance_bonus = spatial_growth - meanfield_growth
```

The candidate was classified as `transient_or_numerical_candidate`. It did not retain positive high-grid covariance support, and the threshold result already failed the validation rule.

## Eco-Evolutionary ODE Model

Step 09A introduced prey defense evolution through total prey density `n`, predator density `w`, and defended-prey frequency `q`.

The ODE equations were:

```text
z = 1/kappa - n - w
r(q) = r_u (1-q) + r_v q
a(q) = a_u (1-q) + a_v q
b(q) = b_u (1-q) + b_v q

dn/dt = n [r(q) z - xi - a(q) w]
dw/dt = w [b(q) n z - (m+s) - mu w]
dq/dt = nu q(1-q) [(r_v-r_u) z - (a_v-a_u) w]
```

The tested parameter values were:

```text
kappa = 0.15
xi = 0.55
r_u = 1
r_v = 0.65
a_u = 1
a_v = 0.35
b_u = 0.08
b_v = 0.02
m = 0.1
mu = 0.2
nu = 0.05
```

The conversion parameters `b_u=0.08` and `b_v=0.02` were adjusted from the setup-note defaults to produce an interior defense-frequency baseline rather than a boundary-pinned `q=1` state.

The baseline state from unstressed burn-in was:

```text
n = 4.8333333
w = 0.64166667
q = 0.67261474
z = 1.1916667
```

The ODE thresholds were:

```text
m_c^{ODE,no evo} = 0.069448242
m_c^{ODE,evo} = 0.16486816
Delta_evo_ODE = 0.095419922
```

In the rescue-window midpoint:

```text
stress window midpoint = 0.1171582
no-evolution predator not persistent
evolution predator persistent
q tail mean = 0.33359896
q change = -0.33901578
```

The Step 09A conclusion was `ODE_indirect_evolutionary_rescue_supported`.

## Spatial Eco-Evolutionary PDE Model

Step 09B added spatial diffusion to the same eco-evolutionary model:

```text
partial_t n = D_n Laplacian(n) + n [r(q)z - xi - a(q)w]
partial_t w = D_w Laplacian(w) + w [b(q)nz - (m+s) - mu w]
partial_t q = D_q Laplacian(q) + nu q(1-q)[(r_v-r_u)z - (a_v-a_u)w]
```

The PDE used zero-flux Neumann boundaries, `D_n = 0.01`, `D_w = 0.01`, and `D_q = 0.005`. The initial state was the same Step 09A burn-in state.

The representative Stage B threshold comparison was:

```text
m_c^{PDE,no evo} = 0.06921875
m_c^{PDE,evo} = 0.11765625
Delta_evo_PDE = 0.0484375
Delta_spatial_no_evo = -0.00022949219
Delta_spatial_evo = -0.047211914
Stage C Delta_spatial_evo = -0.0064306641
```

This initially suggested a lower PDE-evo threshold than the ODE-evo threshold. That was a useful warning signal, but it is not the final interpretation after the later stability and basin analyses.

The PDE evolutionary response remained active near the early threshold estimate:

```text
PDE q tail mean near threshold = 0.31440243
PDE q change near threshold = -0.35821231
```

The spatial mechanism diagnostic did not support amplification in that early threshold framing:

```text
spatial covariance bonus near PDE evolution threshold = -6.2847038e-14
```

## Updated Interpretation After Basin Mapping

PR #4 showed that the apparent PDE-evo threshold was classifier-sensitive.

PR #5 showed unresolved multi-horizon persistence near the sensitive stress range.

PR #6 showed hysteresis and continuation direction dependence: upward continuation could keep a predator-persistent branch at stresses where downward continuation could remain extinct or transient.

PR #7 mapped basin structure and found `pde_evo_bistability_mapped`. In the focused basin scan:

```text
0.141262205: persistent_transient_mixed
0.15: persistent_transient_mixed
0.1584375: bistable_persistent_extinct
0.16486816: bistable_persistent_extinct
0.175: bistable_persistent_extinct
```

Therefore, threshold language must be replaced by stress-regime and basin language. The current result is not a simple suppression result. It is a bistability result.

## Updated Interpretation After Basin-Boundary Mapping

PR #9 quantified basin boundaries in initial-condition space. The scan varied initial defense frequency `q0` and initial predator abundance scale `w0_scale` at stresses `0.1584375` and `0.16486816`.

Both stresses showed `bistable_persistent_extinct` outcomes:

```text
0.1584375: persistent = 20, extinct = 7, transient = 43, unresolved = 0, nonphysical = 0
0.16486816: persistent = 14, extinct = 21, transient = 35, unresolved = 0, nonphysical = 0
```

Persistent basins were concentrated at low-to-intermediate `q0` and modulated by predator initial abundance. High-defense initial states more often entered extinct outcomes at the focused stresses, while many intermediate states remained transient at the current horizon.

Transient outcomes remain common, so long-horizon or targeted continuation remains relevant. The basin-boundary result makes the current endpoint more precise: spatial PDE outcomes should be analyzed through basin structure in `q0`-`w0` space, not as a single scalar persistence threshold.

## Current Conclusion

Current conclusion: ODE prey defense evolution supports indirect evolutionary rescue. In the spatial PDE, the evolutionary rescue response is path-dependent and bistable. Basin entry depends on initial defense frequency and predator abundance, so the spatial model should be analyzed through basin structure rather than a single persistence threshold.
