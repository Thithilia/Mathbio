# Research Note: Spatial Structure Suppresses the Tested Eco-Evolutionary Rescue Window

## Executive Summary

Fixed-defense spatial patterning did not produce robust predator rescue. The fixed-defense Roy-style PDE produced one weak candidate, but it did not survive validation and mechanism checks.

Prey defense evolution produced indirect evolutionary rescue in the well-mixed ODE. Adding spatial diffusion to the eco-evolutionary model did not amplify rescue. In the tested PDE, spatial structure suppressed the rescue window relative to the ODE.

Prey defense evolution can generate indirect evolutionary rescue in the well-mixed model, but the tested spatial PDE suppresses rather than amplifies this rescue.

## Scientific Question

Does spatial structure amplify, suppress, or fail to affect indirect evolutionary rescue generated by prey defense evolution?

The threshold quantities are:

```text
m_c^{ODE,no evo}
m_c^{ODE,evo}
m_c^{PDE,no evo}
m_c^{PDE,evo}
```

The derived threshold differences are:

```text
Delta_evo_ODE = m_c^{ODE,evo} - m_c^{ODE,no evo}
Delta_evo_PDE = m_c^{PDE,evo} - m_c^{PDE,no evo}
Delta_spatial_evo = m_c^{PDE,evo} - m_c^{ODE,evo}
Delta_spatial_no_evo = m_c^{PDE,no evo} - m_c^{ODE,no evo}
```

`Delta_evo > 0` means prey evolution expands the predator persistence range. `Delta_spatial_evo > 0` means spatial structure amplifies evolutionary rescue. `Delta_spatial_evo < 0` means spatial structure suppresses evolutionary rescue.

## Fixed-Defense Spatial Model

Step 07 tested the fixed-defense Roy-style spatial PDE using predator mortality thresholds. The fixed-defense PDE did not show robust `ODE extinct, PDE persistent` behavior. The best weak candidate was the `D_w/D_u=150` case, but Stage D grid escalation did not validate it, so no fixed-defense spatial rescue claim was made.

Step 08 tested whether the `D_w/D_u=150` candidate had a real spatial-growth mechanism. The mechanism diagnostic was:

```text
spatial_growth = mean(w * A)
meanfield_growth = mean(w) * A(mean fields)
spatial_covariance_bonus = spatial_growth - meanfield_growth
```

The candidate was classified as `transient_or_numerical_candidate`. It did not retain positive high-grid covariance support, and the threshold result already failed the validation rule.

## Eco-Evolutionary ODE Model

Step 09A introduced prey defense evolution through total prey density `n`, predator density `w`, and defended-prey frequency `q`.

The ODE equations were:

```text
z = 1/kappa - n - w
r(q) = r_u (1-q) + r_v q
a(q) = a_u (1-q) + a_v q
b(q) = b_u (1-q) + b_v q

dn/dt = n [r(q) z - xi - a(q) w]
dw/dt = w [b(q) n z - (m+s) - mu w]
dq/dt = nu q(1-q) [(r_v-r_u) z - (a_v-a_u) w]
```

The tested parameter values were:

```text
kappa = 0.15
xi = 0.55
r_u = 1
r_v = 0.65
a_u = 1
a_v = 0.35
b_u = 0.08
b_v = 0.02
m = 0.1
mu = 0.2
nu = 0.05
```

The conversion parameters `b_u=0.08` and `b_v=0.02` were adjusted from the setup-note defaults to produce an interior defense-frequency baseline rather than a boundary-pinned `q=1` state.

The baseline state from unstressed burn-in was:

```text
n = 4.8333333
w = 0.64166667
q = 0.67261474
z = 1.1916667
```

The ODE thresholds were:

```text
m_c^{ODE,no evo} = 0.069448242
m_c^{ODE,evo} = 0.16486816
Delta_evo_ODE = 0.095419922
```

In the rescue-window midpoint:

```text
stress window midpoint = 0.1171582
no-evolution predator not persistent
evolution predator persistent
q tail mean = 0.33359896
q change = -0.33901578
```

The Step 09A conclusion was `ODE_indirect_evolutionary_rescue_supported`.

## Spatial Eco-Evolutionary PDE Model

Step 09B added spatial diffusion to the same eco-evolutionary model:

```text
partial_t n = D_n Laplacian(n) + n [r(q)z - xi - a(q)w]
partial_t w = D_w Laplacian(w) + w [b(q)nz - (m+s) - mu w]
partial_t q = D_q Laplacian(q) + nu q(1-q)[(r_v-r_u)z - (a_v-a_u)w]
```

The PDE used zero-flux Neumann boundaries, `D_n = 0.01`, `D_w = 0.01`, and `D_q = 0.005`. The initial state was the same Step 09A burn-in state.

The representative Stage B PDE thresholds and derived differences were:

```text
m_c^{PDE,no evo} = 0.06921875
m_c^{PDE,evo} = 0.11765625
Delta_evo_PDE = 0.0484375
Delta_spatial_no_evo = -0.00022949219
Delta_spatial_evo = -0.047211914
Stage C Delta_spatial_evo = -0.0064306641
```

The PDE evolutionary response remained active near threshold:

```text
PDE q tail mean near threshold = 0.31440243
PDE q change near threshold = -0.35821231
```

The spatial mechanism diagnostic did not support amplification:

```text
spatial covariance bonus near PDE evolution threshold = -6.2847038e-14
```

The Step 09B conclusion was `spatially_suppressed_indirect_evolutionary_rescue`.

## Interpretation

The ODE result shows that prey defense evolution can rescue the predator. The PDE result does not invalidate evolutionary rescue. It says that the tested spatial structure lowers the rescue threshold relative to the well-mixed ODE.

The suppression result is not caused by failure of q to respond; q does decrease. The suppression arises because the spatial PDE supports predator persistence over a smaller stress range than the well-mixed evolving ODE.

The lack of positive spatial covariance means spatial structure does not create an amplification mechanism in the tested setup.

## Current Conclusion

Current conclusion: ODE prey defense evolution supports indirect evolutionary rescue, but the tested spatial PDE suppresses rather than amplifies this rescue.
