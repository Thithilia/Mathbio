# Final Synthesis: Homogeneous Compensation Branch in the Roy Eco-Evolutionary Rescue Model

## Executive Summary

The project began by asking whether spatial structure changes indirect evolutionary rescue in a predator-prey system with prey defense evolution. The final result is not spatial-pattern-mediated rescue. The supported mechanism is homogeneous eco-evolutionary compensation: increased predator mortality stress is compensated by lower prey defense frequency, which increases predator conversion opportunity and preserves a positive predator equilibrium over an interior stress interval.

In the tested spatial PDE, the homogeneous compensation branch is linearly stable to tested Neumann spatial modes. Targeted non-homogeneous perturbations do not produce persistent spatial patterns. Finite-horizon local-defense basin-label changes near boundary states resolve to the matched homogeneous-control basin at longer horizons. Therefore the spatial PDE preserves the homogeneous compensation mechanism rather than creating a new spatial-pattern mechanism.

This conclusion is conditional on the tested linear trade-off formulation and parameter region.

## Original Question

The original motivation was: can spatial structure alter or enable indirect evolutionary rescue in a predator-prey system with prey defense evolution?

The corrected mechanism question became: is the spatial PDE behavior generated by spatial patterning, or does it inherit basin structure from the homogeneous eco-evolutionary reaction system?

## Model Sequence

1. Fixed-defense spatial control.
2. Well-mixed eco-evolutionary ODE.
3. Spatial eco-evolutionary PDE.
4. ODE-PDE mechanism comparison.
5. Compensation-branch derivation.
6. PDE spatial-mode and non-homogeneous perturbation tests.

## Why the Initial Spatial-Threshold Framing Failed

A scalar PDE threshold was initially tested, but long transients, basin dependence, and continuation dependence made scalar threshold language insufficient. Later ODE-PDE comparison showed that the basin structure is mostly inherited from the homogeneous reaction dynamics. Therefore the final explanation should not be written as "PDE threshold lower/higher than ODE threshold." The tested spatial PDE is better interpreted as a spatially extended system preserving homogeneous reaction-level basin structure.

## The Homogeneous Compensation Branch

For the linear trade-off ODE, define

\[
c=\frac{\Delta r}{\Delta a}.
\]

The interior compensation branch is

\[
z^*=\frac{\xi}{r_u-ca_u},
\]

\[
w^*=cz^*,
\]

\[
n^*=\kappa^{-1}-z^*-w^*,
\]

\[
q^*(s)=\frac{(m+s+\mu w^*)/(n^*z^*)-b_u}{b_v-b_u}.
\]

In the current branch, \(n^*\) and \(w^*\) remain approximately fixed while \(q^*(s)\) decreases as stress increases. This decreasing \(q^*\) increases predator conversion opportunity and offsets predator mortality stress at a positive predator equilibrium.

## Existence Conditions

The interior branch exists only if

\[
c>0,
\]

\[
r_u-ca_u>0,
\]

\[
z^*>0,\quad w^*>0,\quad n^*>0,
\]

\[
0<q^*(s)<1.
\]

For the current parameterization, the branch is interior on the stress interval approximately

\[
(-0.1131,\ 0.2324).
\]

## Local Stability by Routh-Hurwitz

For the cubic characteristic polynomial

\[
\lambda^3 + A_1\lambda^2 + A_2\lambda + A_3=0,
\]

Routh-Hurwitz stability requires

\[
A_1>0,\quad A_2>0,\quad A_3>0,\quad A_1A_2>A_3.
\]

PR #22 showed these inequalities hold for all target stresses. Routh-Hurwitz classification agrees with eigenvalue stability. This gives local stability of the homogeneous compensation branch in the tested stress interval.

## PDE Spatial Stability

For the PDE, each Neumann spatial mode evolves under

\[
J_F(U^*)-\lambda_{mn}D.
\]

The zero mode is the ODE stability problem. Nonzero modes test spatial perturbation stability. PR #23 showed all tested nonzero modes are stable under the current diffusion coefficients. The diffusion-ratio linear condition grid found no spatial instability in the tested grid.

## Non-Homogeneous PDE Perturbations

PR #23 tested localized predator patches, localized defense patches, sinusoidal modes, random heterogeneity, and basin-boundary heterogeneity. No persistent spatial pattern was found. Some finite-horizon basin-label changes occurred for localized defense patches near basin-boundary states.

PR #24 followed these cases to longer horizons. Both cases resolved to the matched homogeneous-control basin by longer horizon. Therefore these were finite-horizon transient effects, not persistent spatial-pattern-mediated basin changes.

## Final Mechanism Interpretation

The supported mechanism is homogeneous reaction-level compensation. Prey defense frequency adjusts downward as predator mortality stress increases, preserving a positive predator equilibrium over an interior stress interval. The spatial PDE preserves this homogeneous branch and is linearly stable to tested spatial modes. Targeted non-homogeneous perturbations decay or resolve to homogeneous controls. Spatial-pattern-mediated rescue is not supported in the current parameterization.

## What Is Supported

- ODE indirect evolutionary rescue in the tested parameterization.
- Analytic compensation branch exists under explicit feasibility conditions.
- Current branch matches numerical equilibria to machine precision.
- Current branch is locally stable by Routh-Hurwitz.
- Spatial PDE branch is linearly stable to tested modes.
- Non-homogeneous perturbations do not produce persistent spatial patterns in targeted tests.

## What Is Not Supported

- Spatial-pattern-mediated rescue.
- General claim that spatial structure amplifies or suppresses rescue.
- Global stability.
- Generality over nonlinear trade-offs.
- Generality over all diffusion coefficients.
- Generality over all heterogeneous initial conditions.

## Remaining Caveats

- Linear trade-off only.
- Local structured parameter checks only.
- PDE tests are targeted, not exhaustive.
- No analytical global basin proof.
- No nonlinear trade-off forms tested.
- No biological calibration.

## Recommended Next Scientific Step

The next rigorous step is not more PDE scanning. It is to generalize the compensation-branch conditions to broader trade-off forms or derive parameter-region hypotheses from the existing inequalities. A useful next step is nonlinear, convex, or concave trade-off comparison using the same proposition-based framework.
